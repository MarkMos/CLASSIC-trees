## CLASSIC-trees
A new code to generate dark mater halo merger trees directly from running CLASS, for use with semi-analytical galaxy modeling codes.

## Installation

# Pre-requisites
In order to install CLASSIC-trees one should have a working version of CLASS.

# Downloading
CLASSIC-trees can be downloaded by cloning the GitHub repository:

```bash
$ git clone url
$ cd CLASSIC-trees
```

After that one simply needs to run the following line in the terminal:

```bash
$ python setup.py build_ext --inplace
```
